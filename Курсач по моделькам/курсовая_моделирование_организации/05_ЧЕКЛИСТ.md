# Чек-лист соответствия методическим указаниям ВШЭ («Моделирование организации», 2-е изд.)

Отмечайте пункты по мере выполнения: `[ ]` → `[x]`.

## Оформление отчёта

- [ ] Титульный лист по приложению 1 методички
- [ ] Содержание
- [ ] Введение: до **3 стр.**; актуальность, цель, задачи, структура, источники
- [ ] Основная часть: рекомендуется **3 главы**; параграфы внутри главы — не менее двух, **примерно равный объём**
- [ ] Заключение (кратко, по результатам)
- [ ] Список использованных источников: **ГОСТ**, алфавитный порядок, **не менее 20** источников, в т.ч. **англоязычных**
- [ ] Приложения (при необходимости): анкеты, гайды интервью, громоздкие таблицы
- [ ] Подтверждение оригинальности (приложение 2)
- [ ] Подтверждение равноценности вклада / распределение вклада (приложение 3) — для командной работы
- [ ] Word, поля, шрифт Times New Roman 14, интервал 1,15 — по п. 3.3 методички
- [ ] Все рисунки и таблицы: нумерация, подписи, **ссылки в тексте**; для заимствованных — указание источника
- [ ] Оригинальность текста **не ниже 80%** (консультационный проект)

## Содержание по главам

### Глава 1 — организационные рамки

- [ ] Только **официальная / достоверная** информация
- [ ] **Без анализа и выводов** по методичке (информационный характер)

### Глава 2 — диагностика и методология

- [ ] Обоснование выбора методологии и инструментов под управленческую задачу
- [ ] Описание **способа получения эмпирических данных** (интервью, опрос, вторичные данные и т.д.)
- [ ] Конкретные данные; указание достоверности / ограничений
- [ ] Методы диагностики в приложениях: анкеты, вопросники, документы для интервью (по методичке)

### Глава 3 — модели организации как системы

- [ ] Иерархия «от общего к частному» (обоснована в тексте)
- [ ] Модель процессов **верхнего уровня** (Process landscape / VAD или аналог)
- [ ] При необходимости: организационная структура
- [ ] Детализация: выбранный процесс, модель **AS IS**
- [ ] Анализ проблем, критерии оптимизации
- [ ] Модель **TO BE** с доказательной базой из анализа
- [ ] **Минимум одна модель в нотации BPMN 2.0**
- [ ] **Глоссарий** использованных объектов нотации + краткая **аргументация** выбора BPMN для уровня детализации
- [ ] Предварительная оценка эффекта / организационные требования к изменениям (без выдуманных точных цифр без источника)

## Защита

- [ ] Презентация **12–15 слайдов**
- [ ] Время **10 минут**
- [ ] Готовность отвечать на вопросы по **любой части** работы (в т.ч. для всех членов команды)
