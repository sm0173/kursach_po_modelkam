# Финальная проверка перед сдачей

## Оригинальность

- [ ] Проверка в системе вуза выполнена; результат **не ниже 80%** для консультационного проекта (п. 3.3 методички).
- [ ] Длинные цитаты оформлены по ГОСТ; пересказ с указанием источника.
- [ ] Текст глав — ваша формулировка; фрагменты, сгенерированные ИИ, переработаны и проверены фактами.

## Сквозная логика

- [ ] Задачи из введения закрыты в заключении.
- [ ] Проблемы из главы 2 отражены в анализе AS IS и учтены в TO BE.
- [ ] Номера рисунков и таблиц согласованы с текстом и списком в Word.

## Презентация и защита

- [ ] 12–15 слайдов, тайминг ≤ 10 минут на команду.
- [ ] Файл загружен в Teams / по инструкции учебного офиса.
- [ ] Репетиция ответов на вопросы по всем разделам.

## Документы вуза

- [ ] Подтверждение оригинальности (приложение 2 методички).
- [ ] Подтверждение равноценности вклада (приложение 3) — для командной работы.
