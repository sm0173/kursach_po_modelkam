# Презентация к защите

**Лимиты методички:** **12–15 слайдов**, **10 минут**, PowerPoint (можно экспорт в PDF).

Используйте структуру: [структура_презентации_12-15_слайдов.md](структура_презентации_12-15_слайдов.md).

### Репетиция

- Прогон с таймером; распределение минут по слайдам.
- Подготовка ответов на вопросы по **любой** главе (для командной защиты — каждый участник по своему блоку + общая логика).
