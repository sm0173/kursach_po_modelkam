# Шаблон списка литературы (ГОСТ 7.1-2003, ГОСТ 7.82-2001)

Список в отчёте — **алфавитный** (фамилия автора или заглавие, если автора нет). Иностранные источники — **в конце** общего списка (по методичке). Уточните у методиста программы, если требуется раздельный список RU / EN.

Ниже — примеры **форм** записей (замените на реальные источники; не копируйте примеры как факт существования книг без проверки).

## Книга

```
Иванов И.И., Петров П.П. Моделирование бизнес-процессов : учебник. — Москва : Издательство, 2020. — 350 с.
```

## Статья из журнала

```
Сидоров С.С. Заголовок статьи // Название журнала. — 2021. — Т. 15, № 3. — С. 45–60.
```

## Статья из сборника

```
Козлов К.К. Заглавие главы // Название сборника / под ред. ... — Город : Изд-во, 2019. — С. 120–135.
```

## Ресурс Internet (электронный)

```
Автор (если есть). Название страницы [Электронный ресурс]. — URL: https://... (дата обращения: 09.04.2026).
```

## Англоязычный источник (пример оформления — уточнить по ГОСТ/требованиям кафедры)

```
Dumas M., Rosa M., Mendling J. et al. Fundamentals of Business Process Management. — Springer, 2018. — XX p.
```

## Чек перед сдачей

- [ ] Каждая позиция в списке имеет хотя бы одну **ссылку в тексте** или явно указана как использованная при подготовке (по методичке — все использованные источники включаются в список).
- [ ] Для веб-страниц указана **дата обращения**.
- [ ] Не менее **20** позиций, из них **несколько англоязычных**.

## Работа с ИИ

Попросите агента: «Преобразуй этот список в алфавитный порядок по ГОСТ» — но **проверьте** каждое поле по первоисточнику.
