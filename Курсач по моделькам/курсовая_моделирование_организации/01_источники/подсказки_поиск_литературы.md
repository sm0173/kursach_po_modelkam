# Подсказки для набора ≥20 источников (глава 1 + теория)

Используйте как ориентиры для поиска в elibrary.ru, Google Scholar, сайтах издательств. **Проверяйте** реквизиты по первоисточнику перед внесением в список литературы.

## Тематические блоки (примеры направлений запросов)

1. **BPM / BPMN:** `BPMN 2.0 specification`, `business process management`, `процессный подход управление`.
2. **Моделирование организации:** `enterprise modelling`, `ARIS`, `EPC`, `value-added chain`.
3. **Диагностика процессов:** `process mining`, `process assessment`, `зрелость процессов`, `SIPOC`, `RACI`.
4. **Стратегия и показатели:** `balanced scorecard`, `KPI`, `стратегическая карта`.
5. **Отраслевой контекст:** добавьте 2–4 источника по **вашей** отрасли (отчёты, аналитика — с датой и URL).

## Англоязычные источники

Цель методички — наличие **англоязычных** позиций. Подойдут статьи из журналов по BPM, главы из учебников Springer/Wiley **после** проверки библиографии.

## Работа с ИИ в Cursor

Запрос: «Составь таблицу из 15 реальных монографий и статей по BPMN и диагностике процессов с DOI или издательством» — затем **вручную** удалите несуществующие записи и дополните до 20 проверенными.
