## Входящие файлы

**`00_Входящие/`** — сюда кидаешь любой новый файл.  
Агент читает, извлекает полезное, обновляет СТАТУС и КОНТЕКСТ, перемещает файл в нужную папку.  
Протокол обработки: `00_Входящие/README.md`.
