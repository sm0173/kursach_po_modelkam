## Справочник кодов

### Проблема 1 — доступность / ETA / SLA

| Код | Когда ставить |
|-----|----------------|
| `отказ_недоступность` | Клиент не может оформить заказ, адрес/доставка недоступны, заказ отклонён |
| `альтернативная_точка` | Упоминается другая точка, другой слот, самовывоз или отсутствие такой альтернативы |
| `задержка` | Опоздание, долгое ожидание, «ждали час» |
| `ETA_несовпадение` | В приложении одно время, по факту другое; статус не совпал с реальностью |
| `ETA_неясен` | Нет понятного срока, только статусы, непонятно когда ждать |
| `нет_уведомления` | Не было push/SMS о задержке |
| `холодная_еда` | Еда остыла / несвежая (часто следствие задержки) |
| `отмена` | Отмена заказа из‑за ожидания |
| `зона_доставки` | Адрес недоступен, срезали зону, правило 60 минут |
| `компенсация` | Скидка, сертификат, бесплатная пицца за опоздание |

### Проблема 2 — рекламации

| Код | Когда ставить |
|-----|----------------|
| `поддержка_долго` | Долго ждали ответ / не ответили |
| `поддержка_непонятно` | Непонятно, куда жаловаться; совет не помог |
| `чат_оператор` | Чат, бот, эскалация на оператора |
| `компенсация` | Жалоба на размер/отсутствие компенсации |
| `повторная_проблема` | Повторный сбой на той же точке |

*Один отзыв может иметь 2–3 кода. Колонки «ETA» / «Рекламации» — да, если код относится к линии проблемы.*

## Как продолжать сбор (№ 28–120)

1. **Google Play / App Store** — фильтр 1–2★, поиск: `опозд`, `ждал`, `час`, `холод`, `поддерж`, `чат`.  
2. **IRecommend** — «Додо Пицца» + «доставка» + отзывы с минусами.  
3. **Яндекс Карты / 2ГИС** — отзывы на пиццерии в вашем городе, тег «доставка».  
4. Для каждой строки: URL, фрагмент 1–2 предложения, коды, дата (если видна).
