## Справочник кодов

### Проблема 1 — ETA / доступность

| Код | Когда ставить |
|-----|----------------|
| `отказ_недоступность` | Доставка недоступна, заказ не приняли |
| `альтернативная_точка` | Нет / нужна другая точка или слот |
| `задержка` | Опоздание, долгое ожидание |
| `ETA_несовпадение` | Обещали одно время — по факту другое |
| `ETA_неясен` | Только этапы, нет понятного срока |
| `нет_уведомления` | Не было push о задержке |
| `холодная_еда` | Еда остыла (следствие задержки) |
| `зона_доставки` | Адрес вырезан из зоны |

### Проблема 2 — рекламации

| Код | Когда ставить |
|-----|----------------|
| `поддержка_долго` | Долгий ответ / тишина |
| `поддержка_непонятно` | Непонятно, куда жаловаться |
| `чат_оператор` | Бот → оператор |
| `компенсация` | Промокод / скидка |
| `повторная_проблема` | Повтор на той же точке |
