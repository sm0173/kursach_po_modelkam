# Сырые заметки

Помещайте сюда файлы с записями интервью, опросов и выгрузок.

| Файл | Что это |
|------|---------|
| **[примеры_отзывов_учебные.md](примеры_отзывов_учебные.md)** | 15 синтетических отзывов для тренировки кодов (не реальный сбор) |
| `отзывы_агрегаты_ГГГГММДД.md` | Агрегаты после разметки реальных отзывов |
| `YYYY-MM-DD_роль_респондента_условно.md` | Заметки интервью |

**Правило:** для приложения к работе используйте **анонимизированные** версии (без ФИО и идентифицирующих деталей, если нет согласия).

Имена новых файлов: `YYYY-MM-DD_роль_респондента_условно.md` (например, `2026-04-09_клиент_доставки_01.md`).
