# Сырые заметки

Помещайте сюда файлы с записями интервью и опросов.

**Правило:** для приложения к работе используйте **анонимизированные** версии (без ФИО и идентифицирующих деталей, если нет согласия).

Имена файлов: `YYYY-MM-DD_роль_респондента_условно.md` (например, `2026-04-09_менеджер_продаж_01.md`).
